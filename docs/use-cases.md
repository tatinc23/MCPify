# Top 10 Use Cases for MCP Server Templates

The highest-value verticals where non-technical SMBs benefit from exposing their business to AI agents via paid MCP tools.

| # | Template | Core MCP Tools | x402 Revenue Model | Phase 2 Add-Ons |
|---|---|---|---|---|
| 1 | **Restaurant / Cafe** | Menu lookup, table reservations, takeout ordering, hours | $0.01–$0.10 per call | Stream (live kitchen cam), R2 (menu photos), AI (marketing copy) |
| 2 | **Salon / Spa / Barbershop** | Service catalog, stylist availability, booking, pricing | $0.02–$0.05 per call | Stream (live chair availability), AI (style recommendations) |
| 3 | **Real Estate Listings** | Property search, schedule viewing, neighborhood data, mortgage calc | $0.05–$0.25 per call | R2 (property photos/tours), AI (listing descriptions), Stream (live open house) |
| 4 | **Auto Repair Shop** | Service quotes, appointment booking, parts lookup, status check | $0.05–$0.15 per call | R2 (repair photos), AI (diagnostic suggestions from OBD codes) |
| 5 | **Fitness Studio / Gym** | Class schedule, trainer booking, membership pricing, capacity check | $0.02–$0.10 per call | Stream (live class feed), AI (personalized workout plans) |
| 6 | **Dental / Medical Clinic** | Appointment booking, insurance verification, service pricing, intake forms | $0.05–$0.20 per call | R2 (intake document storage), AI (post-visit care instructions) |
| 7 | **Home Services (Plumber/Electrician/HVAC)** | Service request, emergency dispatch, quote generation, availability | $0.10–$0.50 per call | R2 (before/after photos), AI (troubleshooting guides) |
| 8 | **Event Venue / Catering** | Availability calendar, pricing tiers, capacity, catering menu, booking | $0.05–$0.25 per call | Stream (virtual venue tour), R2 (event photos), AI (event planning suggestions) |
| 9 | **Legal / Accounting Practice** | Consultation booking, service catalog, document intake, FAQ | $0.10–$0.50 per call | R2 (secure document storage), AI (document summarization) |
| 10 | **E-commerce / Boutique Retail** | Product catalog, inventory check, order placement, shipping status, returns | $0.01–$0.10 per call | Stream (live shopping), R2 (product images), AI (product recommendations) |

## Selection Criteria

- **Non-technical owner** — can't build an API themselves
- **High agent query volume** — AI agents will call these tools frequently (bookings, lookups, quotes)
- **Clear monetization** — each tool call has intrinsic value worth paying for
- **Phase 2 upside** — benefits from Stream, R2, or Workers AI add-ons
- **Clone-ability** — same schema pattern (catalog + bookings + payments) maps across verticals
